# 🔄 Unit Converter Pro

A modern, feature-rich **Unit Converter** desktop application developed using **C#** and **Windows Forms**. The application provides fast and accurate conversions across multiple unit categories with a clean interface, advanced search functionality, and dark mode support.

## ✨ Features

* 🌙 Dark Mode support
* 🔍 Advanced unit search and filtering
* ⚡ Real-time conversion
* 🔄 Swap source and destination units
* 📂 Multiple unit categories
* 🎯 Accurate conversion calculations
* 🖥️ Clean and user-friendly interface
* 📱 Responsive window layout
* 🧹 Well-structured and maintainable code
* 🚀 Fast and lightweight performance

## 📦 Supported Categories

* Length
* Weight
* Temperature
* Area
* Volume
* Time
## 🚀 How to Run the Project

### Requirements

Before running the project, make sure you have:

* **Windows Operating System**
* **Visual Studio 2022** (or later)
* **.NET Desktop Development workload** installed in Visual Studio

### Steps

1. Clone or download this repository:

```bash
git clone https://github.com/dinazangeneh/unit-converter-csharp.git
```

2. Open the project folder.

3. Open the solution file:

```text
UnitConverter.sln
```

with **Visual Studio**.

4. Wait for Visual Studio to restore the project dependencies.

5. Press:

```text
F5
```

or click the **Start ▶ button** to build and run the application.

### Alternative

If you only want to use the application, download the latest release (if available), extract the files, and run the `.exe` file.
